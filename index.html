④ 下載 + 上架

上架到 LINE Creators Market 完整步驟

① 註冊 / 登入 Creators 帳號

  1. 到 creator.line.me 點右上「登入」用 LINE 帳號授權。
  2. 第一次會要填「創作者名稱」「Email」，同意條款。

② 建新貼圖

  1. 進「個人頁面 → 我的貼圖」，點 「新增 → Sticker（貼圖）」。
  2. 進入後分 3 個 tab 都要填：
    • 貼圖介紹（文字 metadata）— 見下面 ③
    • 貼圖管理 → 圖片編輯（檔案）— 見 ④
    • 販售資訊（價格 / 地區 / 活動 tag）— 見 ⑤

③ 文字 metadata（手填）

  • 貼圖標題：40 字內
  • 貼圖簡介：160 字內
  • 創作者名稱：50 字內
  • 著作權標示（Copyright）：50 字內，限英數，例如 © 2026 yourname

④ 上傳圖檔（兩種方法擇一）

  • 方法 A：整包 ZIP 拖入（推薦）
    「貼圖管理 → 圖片編輯」頁面有「上傳壓縮檔 (ZIP)」按鈕， 把這裡下載的 ZIP 整個拖進去，main.png / tab.png / 01.png~08.png 會自動分配到對應欄位。我們輸出的檔名跟尺寸完全照 LINE 規格、ZIP 才幾十 KB（限制 60MB）。
  • 方法 B：一張一張上傳
    解壓後，把 main.png 拖到「主要圖片」欄、tab.png 拖到「聊天室標籤」欄、 01.png~08.png 依序拖到「貼圖 1」~「貼圖 8」欄。

⑤ 販售資訊

  • 銷售區域：選「台灣」或「全球」（建議全球曝光大）
  • 主要使用語言：繁體中文
  • 價格：靜態貼圖最低 NT$30（眼淚製造機特輯最低 NT$40、台味 NT$60 固定價）
  • 特輯活動 tag（投稿活動才需要）：在「販售資訊」最下面有「特輯活動」勾選， 勾你對應的活動（例如「眼淚製造機表情貼特輯」）

⑥ 送審

  • 三個 tab 都填好後，點「申請販售」。
  • 通常 1-7 天內 LINE 會回審核結果（通過 / 退件理由）。
  • 通過後狀態變「可發售」，再點「上架」就會出現在 LINE STORE 上。
  • 每筆銷售你拿 50%（其他扣 LINE 平台費 + 三方支付手續費）。

⚠ 常見退件原因：
• 肖像權：用他人照片但沒授權（→ 確認你就是圖中本人，或已書面授權）
• 商標：包含 LINE 自家、品牌、藝人、卡通角色（→ 用自己原創角色）
• 文字過多：整片都是字（→ 我們已限制每格只 1 句、視覺為主，通常不會中）
• 解析度 / 透明背景不符（→ 本工具已輸出標準 370×320 透明 PNG，正常不會中）
• 重複申請：同一組貼圖反覆送（→ 退件後改完再送）
